Chat Widget Demo

Esta página demonstra o widget embeddable. O botão de chat aparece no canto inferior direito.

Como instalar em qualquer site:


<script>
  window.ChatWidget = { key: 'SEU_WIDGET_KEY' };
</script>
<script src="https://seudominio.com/widget.js" async></script>

O widget persiste a conversa via cookie. Se o visitante fechar e reabrir o browser, o histórico é mantido. Quando o agente responde offline, o cliente recebe um email.
